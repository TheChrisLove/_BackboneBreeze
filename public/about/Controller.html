Class: Controller

Controller

Controller The base controller class from which all controllers extend. Provides methods for managing views, layouts, accessing data in and across views, and handling route actions. Controllers are initialized within Info model or within Router on route match. Controller information for an extand module is available through Info.

Accessor for retrieving/initializing views. If the view is currently active or persisted in this.get('views'), that view object will be returned. Otherwise, a new view object will be created from the definition passed in and returned.

Name Type Description
name string optional

The name of the view, only needed if you plan on persisting the view.
View View optional

A definition of the view. If you are only checking on the existance of an existing view, this would be optional.
model Model optional

A model to pass to view initialization, should the view need to be initialized.
options object optional

Backbone.View options to be passed to the view should the view need to be initialized.
Returns:
